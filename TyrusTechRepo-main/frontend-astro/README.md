# Digitization Platform Frontend

This repository contains the frontend for the Digitization Platform, built using Astro.

## Getting Started

To get started with the frontend, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd digitization-platform/frontend-astro
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open your browser**:
   Navigate to `http://localhost:3000` to view the application.

## Project Structure

- **src/pages**: Contains the pages of the application.
- **src/components**: Contains reusable components.
- **src/layouts**: Contains layout components for structuring pages.
- **src/styles**: Contains global styles for the application.
- **src/types**: Contains TypeScript type definitions.

## Build

To build the application for production, run:

```bash
npm run build
```

This will generate static files in the `dist` directory.
