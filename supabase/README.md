# Supabase Setup for Digitization Platform

This directory contains the configuration and setup files for the Supabase database used in the Digitization Platform project.

## Configuration

- **config/config.toml**: This file contains the configuration settings for the Supabase project, including database connection details and environment settings.

## Migrations

- **migrations/001_init.sql**: This file includes the SQL commands to initialize the database schema. It is essential to run this migration to set up the necessary tables and relationships.

## Functions

- **functions/README.md**: This file provides documentation for any serverless functions that may be implemented in the Supabase project. It includes usage instructions and examples.

## Getting Started

To set up the Supabase environment, follow these steps:

1. Ensure you have a Supabase account and create a new project.
2. Configure the `config/config.toml` file with your project details.
3. Run the SQL migration script to initialize your database schema.
4. Refer to the functions documentation for any serverless functions you may want to implement.

For further details, please refer to the main project documentation.
